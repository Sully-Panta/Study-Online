¿Qué vas a aprender en este curso?
En este curso aprenderás a utilizar HTML y CSS, los cimientos de la web. l primero de ellos hace referencia al lenguaje que se utiliza para decirle al navegador lo que es cada una de las partes de la web y con el segundo, damos formato y forma a nuestra web. Con el aprendizaje de ambas tecnologías conseguirás crear desde páginas webs a aplicaciones móviles o videojuegos, diseñando la interacción perfecta entre nterfaz y usuario.
Hasta dónde llegaremos
Al terminar el curso habrás aprendido a crear tu propio portfolio paso a paso comprendiendo cuáles son las diferentes partes de un documento HTML, cómo organizar la información, recursos y consejos para realizar un buen diseño, qués es un modelo de cajas y cómo crearlo, además de todos los detalles que contiene una web. Gracias a los ejercicios, retos y proyecto del curso, podrás asimilar todo lo indicado de una forma divertida y práctica.
